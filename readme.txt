beatmaniaIIDX INFINITASのリザルト画像をファイルに保存します。
リザルトをすぐ閉じずに、1秒くらいは表示させ続けてください。
保存したリザルト画像はresultsに保存されます。

---保存されたリザルトを都度表示する---
リザルト画像をファイルに保存される度、ウィンドウ上にリザルトを表示します。
処理落ちしてINFINITASに悪影響を与える可能性があります。

---更新があるときのみリザルトを保存する---
クリアタイプ・DJレベル・スコア・ミスカウントのいずれかにNEW RECORDがついているときのみ
リザルトを保存します。

---ライバルを隠して別に保存する---
リザルトを選択してボタンを押すと
ライバルとライバル挑戦状にぼかしを入れた画像をfilteredに保存します。
