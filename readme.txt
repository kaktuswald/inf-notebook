beatmaniaIIDX INFINITASのリザルトからプレイデータを記録します。

リザルトをすぐ閉じずに、1秒くらいは表示させ続けてください。
ミッションやビット獲得のカットインが表示された場合は、表示が消えてから1秒程度の表示が必要です。

保存したリザルト画像はresultsに保存されます。
ぼかしをかけて保存したリザルト画像はfilteredに保存されます。
記録されたプレイデータはrecordsに保存されます。

---機能---
リザルトから自身の記録を記録します。
選曲中に選択している曲の自己ベストを記録します。
リザルト画像を自動でファイルに保存することができます。
リザルト画像のライバル欄にぼかしを入れることができます。
これまでのプレイのクリアタイプ・DJレベル・スコア・ミスカウントを見ることができます。
画像を保存していた場合は画像も確認できます。
スコアとミスカウントの推移を見ることができます。
プレイ済みの曲を検索して、プレイ履歴を見ることができます。
譜面ごとにグラフを出してスコアとミスカウントの変動を見ることができます。
記録データをCSVに出力することができます。
OBSと連携することができます。
